Axe Public Intent API

Integrate Discord Rich Presence into your Android applications. Axe handles the gateway communication, allowing you to update user status with simple Broadcast Intents.

Setup / Prerequisites

Permission

Add the Axe RPC permission to your AndroidManifest.xml:

<uses-permission android:name="com.my.axe.PERMISSION_RPC" />

Package Visibility

For Android 11 (API 30) or higher, include Axe in your queries:

<queries>
    <package android:name="com.my.axe" />
</queries>

Available Intent Actions

Action Description
com.my.axe.UPDATE_PRESENCE Update or start a Discord presence session.
com.my.axe.CLEAR_PRESENCE Stop the active presence session.

Supported Extras

Sent via UPDATE_PRESENCE.

Key Type Description
activity_nameStringMain title of the activity
detailsStringSecondary status line
stateStringThird status line
large_imageStringPublic image URL
small_imageStringPublic image URL (overlay)
timestamp_startLongStart time in ms
activity_typeStringplaying, listening, etc.
source_packageStringYour app's package name

Kotlin Example

fun updateAxePresence(context: Context) {
    val intent = Intent("com.my.axe.UPDATE_PRESENCE").apply {
        `package` = "com.my.axe"
        putExtra("activity_name", "Kingdom of Kotlin")
        putExtra("details", "Battling Bugs")
        putExtra("large_image", "https://example.com/asset.png")
        putExtra("timestamp_start", System.currentTimeMillis())
        putExtra("activity_type", "playing")
        putExtra("source_package", context.packageName)
    }
    context.sendBroadcast(intent)
}

Java Example

public void updateDiscordPresence(Context context) {
    Intent intent = new Intent("com.my.axe.UPDATE_PRESENCE");
    intent.setPackage("com.my.axe");
    intent.putExtra("activity_name", "Java Legends");
    intent.putExtra("timestamp_start", System.currentTimeMillis());
    intent.putExtra("activity_type", "playing");
    intent.putExtra("source_package", context.getPackageName());
    context.sendBroadcast(intent);
}

Best Practices

  • Rate Limiting: Enforced at 15 seconds per source.
  • Image Hosting: Use stable, direct public URLs.
  • Lifecycle: Clear presence on application exit.

Clear Presence

val intent = Intent("com.my.axe.CLEAR_PRESENCE").apply {
    `package` = "com.my.axe"
}
context.sendBroadcast(intent)
